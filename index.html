How to Use Class Points

1. Add a Student: Enter a student's name in the input field at the bottom and click "Add Student."

2. Update Points: Use the buttons (+1, +5, +10) below each student's name to adjust their points. You can also type a number directly into the input box.

3. Bulk Update Points: Select one or more students by clicking their avatar or name, then use the buttons at the top of the page (+1, +5, +10) to update points for all selected students.

4. Select/Deselect All: Use the "Select/Deselect All" button to quickly select or deselect all students.

5. Reset Points: Click "Zero" to reset points for all selected students (or all students if none are selected).

6. Sort Students: Use the "Sort" button to sort students alphabetically by name, or "Leaderboard" to sort by points.

7. Backup/Restore: Use "Backup" to export student data as a CSV file. Use "Import" to restore data from a CSV file.

8. Celebrate: Click "Celebrate!" to launch confetti and celebrate your class's achievements!

9. Undo: Use the "Undo" button to revert the last action (up to 10 actions).
